Forgotten Memories

by Xodius Studios

Release Date: June 2023

Roles: Team Lead Game Designer Tools Designer Rigging UI

Tools: Unreal 4.27 Visual Studio Enterprise Blender Gimp

  • Project managing and coordinating of team members in an online setting to create this hack-n-slash adventure.
  • Developed the tutorial level with traps and text pop-ups to teach the player how to play before embarking on their adventure.
  • Created player character using C++ and Blending Trees for animation and attack combos with a smooth attack feel.
  • Modified Assets in Blender and Gimp to allow for Skeletal Rigging with new animations.
  • Created Interactive Menus with sounds/backgrounds for the Title and Victory/Death Screen, which blends well with the toonish atmosphere.
